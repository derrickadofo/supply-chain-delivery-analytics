# supply-chain-delivery-analytics
Supply chain analytics project analyzing e-commercedelivery performance, logistics delays, and operational efficiency using Python, SQL, and BI dashboards.

## Project Overview
This project analyzes logistics and delivery performance using a large real-world e-commerce dataset. The objective is to identify operational bottlenecks, measure delivery reliability, and generate data-driven insights that could improve logistics efficiency and customer satisfaction.
The analysis focuses on key operational questions such as delivery delays, seller performance, shipping cost efficiency, and regional logistics patterns.
The project demonstrates practical analytics skills used in business intelligence and operations analytics roles.

## Business Questions
Which factors drive delivery delays in an e-commerce supply chain?
Which sellers, regions, or product categories contribute most to late deliveries?
How does shipping cost relate to delivery speed and logistics efficiency?
What operational improvements could increase on-time delivery rates?
